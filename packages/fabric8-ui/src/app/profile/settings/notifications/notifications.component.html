Delivery Methods

System Widget Notifications

These will only be presented in the notification drawer, represented by the [] icon.

Email Notifications

These will only be sent to the primary email listed in your user profile.

Global Notification Triggers

Assigned to an issue

You will receive notifications for all activity or any issues that you are assigned to.

Status changes

You will receive notifications when a status changed on a work item that you are collaborating on.

Comments

You will receive notifications for all comments that occur after any comment that you have been added to a work item. This includes any changes a user may make to an existing comment, or if you have been tagged in a comment or on a work item.
